# GigaChat Agent - Комплексная система многоагентных интеллектуальных помощников

Потестировал различные модели гигачата, в итоге вывел примерный разброс, под какие задачи и что можно брать.

Важно задавать вопросы максимально абстрактные, на которые нет однозначного ответа

| Модель      | Температура | Обоснование                                         |
| ----------- | ----------- | --------------------------------------------------- |
| MAX или PRO | 0.0-0.3     | Стабильность критична, нужна предсказуемость оценок |
| STANDARD    | 0.7-0.97    | Разные углы зрения, должны отличаться друг от друга |
| PRO         | 0.5-0.7     | Качественный анализ, но проверенная логика          |
| MAX         | 0.3-0.5     | Самая развернутая, хорошая структура                |
